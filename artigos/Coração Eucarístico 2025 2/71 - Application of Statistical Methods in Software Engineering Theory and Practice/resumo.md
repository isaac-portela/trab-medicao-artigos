# Resumo

## Identificação
- Título: Application of Statistical Methods in Software Engineering: Theory and Practice
- Aluno: Lucas Randazzo
- Grupo: Coração Eucarístico 2025/2
- Arquivo: artigo.pdf
- Ano: 2020
- Fonte / Evento / Periódico: arXiv
- DOI / URL: https://arxiv.org/abs/2006.15624

## Objetivo

Apresentar técnicas estatísticas aplicáveis a estudos experimentais em Engenharia de Software e oferecer uma abordagem prática para escolher métodos de análise de dados. O artigo combina conceitos básicos com exemplos e um fluxo de decisão.

## Problema investigado

Pesquisas em Engenharia de Software precisam validar resultados, mas a escolha inadequada de testes estatísticos pode gerar conclusões frágeis. O problema é orientar pesquisadores na seleção de métodos compatíveis com tipo de dado, desenho do estudo e pergunta de pesquisa.

## Método
- Tipo de estudo: artigo metodológico com abordagem prática.
- Dados / amostra: dados reais de projetos de software usados para demonstrar técnicas estatísticas.
- Procedimento: apresentação de conceitos de estatística, discussão de técnicas e uso de uma árvore de decisão para indicar métodos de análise.
- Métricas, modelos ou instrumentos utilizados: estatística descritiva, estatística inferencial, árvore de decisão para escolha de testes, variáveis de projetos de software.
- Técnicas estatísticas, quando houver: testes estatísticos e métodos de análise de dados adequados a diferentes situações; o texto enfatiza a seleção correta conforme características dos dados.

## Principais resultados
- Resultado 1: O artigo mostra que a análise de dados em Engenharia de Software deve passar por coleta, descrição e inferência.
- Resultado 2: A árvore de decisão facilita a escolha de métodos estatísticos por pesquisadores com menor familiaridade estatística.
- Resultado 3: Exemplos com dados de projetos tornam a discussão operacional e aproximam teoria estatística de prática experimental.

## Contribuições

A contribuição principal é traduzir técnicas estatísticas para o contexto de Engenharia de Software, reduzindo a distância entre desenho experimental e análise correta. O artigo também reforça que validade empírica depende de análise compatível com os dados.

## Limitações

Embora o PDF tenha texto bastante extraível, alguns trechos apresentam separação irregular de caracteres. O trabalho tem objetivo instrucional e não pretende esgotar todos os métodos estatísticos possíveis.

## Relação com a disciplina
Conexão com:
- Medição em Engenharia de Software: depende da definição e coleta correta de variáveis mensuráveis.
- Análise de Medições de Software: é diretamente relacionado à escolha e aplicação de técnicas estatísticas.
- Experimentação em Engenharia de Software: apoia validação de hipóteses, comparação de tratamentos e interpretação de resultados experimentais.

## Potencial para o trabalho final

É uma referência central para qualquer trabalho final que colete dados e precise escolher testes, sumarizações ou análises estatísticas.

## Citação curta

A validade de um estudo em Engenharia de Software depende de escolher métodos estatísticos coerentes com os dados e com a pergunta de pesquisa.
